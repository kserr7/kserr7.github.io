My Skills

Adobe Photoshop

I use Photoshop to create my digital art. I've created sprite sheets, my logo, and many personal projects that mainly consist of character concept art.

Adobe Animate

I create intro animiations for my freshman year projects to help tell some of the story. I also use it to help create sprite sheets so the final animation in-game is smooth.

Adobe Premiere Pro

For my first video game trailer and other small projects, I used Premiere Pro to edit clips I have recorded, as well as compositing green screen.

Unity

Unity is the game engine the GIMM program at BSU has us learn and use. I've created 2D and 3D games and interactive experiences with C# code. I've also created an AR experience as a team with other GIMM classmates.

Web Design

I've learned HTML, CSS, and Javascript to create mobile friendly websites. I've also used Bootstrap, SQL, Node.js, D3, JSON, and React intermixed where assigned to create dynamic web pages.

Visual Storytelling

With character design, my favorite kind of art, it's important to give an idea of their personality by their visual apperance. I use visual storytelling to direct the choices for my designs
